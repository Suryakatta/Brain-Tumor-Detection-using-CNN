# Brain Tumor Detection using CNN

This project uses Deep Learning and CNN to detect brain tumors from MRI images.

## Features
- MRI image classification
- Tumor / No Tumor prediction
- CNN model using TensorFlow/Keras
- Flask web application
- Confusion matrix evaluation

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- Flask
- NumPy
- Matplotlib

## Dataset
Brain MRI Images for Brain Tumor Detection

## Accuracy
Achieved around 90% test accuracy.

```bash
pip install -r requirements.txt
python app.py
